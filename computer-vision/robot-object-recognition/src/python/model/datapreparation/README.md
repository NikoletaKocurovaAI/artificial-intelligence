# Data preparation

## Project set-up

### Virtual environment

#### Create

Create virtual environment for Mac OS:

```
virtualenv data-prep-venv
```

#### Activate

Activate virtual environment for Mac OS:

```
source ./data-prep-venv/bin/activate
```

### Install packages

```
pip install -r requirements-dev.txt
```

## Run main
```
python main.py
```
